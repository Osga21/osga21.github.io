-Picromaker (2016)

5x5 puzzle maker for the 3ds homebrew game Picroxx! More of a proof of concept than anything else, but it actually does what it's supposed to do. Exports the code for a level. You need python 2.7 to run this. Not even close to finished.

-EZ-FlashIV Skin "Dark Flat"(2017)

Simple modern skin for the popular GBA flashcart, originally made for me but I decided to release it because the EZ-Flash community is pretty much dead, even though the cartridge is still being sold.

Built on the latest kernel(1.78). The original files include the photoshop files, the skin image files and the skin builder along with the kernel. The skin builder was made by libertyernie
